    ## 手写幻灯片渲染器！！

    语法和 UOJ 一样，是 YAML + Markdown。

    原理很简单，先用 YAML 解析一遍，然后用 Markdown 渲染，拆成若干个 div 元素，最后拼一起。

    有没有发现样式和 UOJ 很像，因为照着颜色选的。。。

    开发中。。很多样式很丑，丑的很别致，就懒得改了

    [点我试试](/app/slide.html)

- |
    ### 语法教程

    ```yaml
    ---
    - |
    这里写第一页幻灯片的 Markdown
    - |
    这里写第二页幻灯片的 Markdown
    - |
    这里写第三页幻灯片的 Markdown
    - children:
        - |
            哇咔咔竖着第一页
        - |
            哇咔咔竖着第二页
    ...
    ```
